The STK1000 has (number) jumpers that can be set to configure the board.
For this assignment, the following settings were set for the jumpers:

\begin{table}
\begin{tabular}{|l|l|}

SW1 &
Set to SPI0.
\\

SW2 &
Set to PS2A/MMCI/USART1.
\\

SW3 &
Set to SSC0/PWM[0,1]/GCLK.
\\

SW4 &
Set to GPIO.
\\

SW5 &
Set to LCDC.
\\

SW6 &
Set to GPIO.
\\

JP4 &
Set to "INT. DAC".
\\

JP5 &
Set to "INT. DAC".
\\

\end{tabular}
\end{table}
